# Prettify JSON

Предназначен для human-readeble вывода json файлов. 

# Quickstart
Запуск скрипта на Linux и python3.5

```#!bash

$ python pprint_json.py <path to file>
{
    key1: value1,
    key2: value2
}

```
Аналогично для Windows
